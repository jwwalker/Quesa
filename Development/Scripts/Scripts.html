Quesa Scripts

This directory contains various Perl scripts used in the development of Quesa.

Script Description
Skeleton Generated stub code from the QD3D headers. Was used to to generate the initial Quesa source tree.
Headeriser Generated the Quesa headers from the Quesa source. Was used to generate the initial Quesa headers.
Status Generates the status table for the docs.
HeaderDoc Generates the API level documentation from the headers. A customised version of Apple's HeaderDoc script. Licensed under the APSL.
BuildRelease Generates the release archives. Must be run under MacPerl, with the cpan-mac modules installed (see the script header for more details).

To run these scripts, you will need a copy of Perl
