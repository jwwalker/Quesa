TriMesh

Summary

The TriMesh is the core vertex-based geometry in Quesa, and is designed for optimal rendering performance at the cost of flexibility.

Notes

Unlike QD3D, Quesa's TriMesh object will automatically normalize any triangle and vertex attributes assigned to the triangles or vertices in the TriMesh. Applications which submit immediate mode TriMesh data structures should ensure that these normals are also normalized, as renderers expect all TriMesh normals to be pre-normalized.

Performance Tips

Notes

For objects which are not transparent, backface culling may be turned on to remove those polygons which will not be visible in the final scene. To perform backface culling, renderers require a normal for each polygon. To reduce the overhead of calculating these normals, you should supply triangle normals for all polygonal geometry if you intend to use backface culling.
