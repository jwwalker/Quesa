Q3Camera_GetType

TQ3ObjectType  
Q3Camera_GetType (
    TQ3CameraObject               camera
);

Returns the type of a camera.

Returns kQ3CameraTypeOrthographic, kQ3CameraTypeViewAngleAspect, or kQ3CameraTypeViewPlane. Returns kQ3ObjectTypeInvalid if the camera type is unknown.

Parameters

NameDescription
cameraThe camera to test.
Result: The type of the camera object.
