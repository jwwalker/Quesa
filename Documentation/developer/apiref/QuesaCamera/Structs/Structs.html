TQ3CameraPlacement

typedef  struct  {
  TQ3Point3D  cameraLocation;
  TQ3Point3D  pointOfInterest;
  TQ3Vector3D  upVector;
}  TQ3CameraPlacement;

Describes the location and orientation of a camera. All points are in world-space coordinates.

Fields

NameDescription
cameraLocationThe location of the camera.
pointOfInterestThe point at which the camera is aimed.
upVectorThe up vector for the camera. This vector must be normalised and perpendicular to the viewing direction of the camera. This vector is transformed to the y axis of the viewing plane.

TQ3CameraRange

typedef  struct  {
  float  hither;
  float  yon;
}  TQ3CameraRange;

Describes the hither and yon clipping planes of a camera.

Fields

NameDescription
hitherThe distance from the camera to the near clipping plane. This value must always be more than 0.
yonThe distance from the camera to the far clipping plane. This value must always be more than the hither field, although if it is too large then artifacts may be visible during rendering.
