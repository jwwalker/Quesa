TQ3FisheyeCroppingFormat

Type of cropping for a fisheye camera 
typedef enum TQ3FisheyeCroppingFormat
{
  kQ3FisheyeCroppingFormatCircular = 0,
  kQ3FisheyeCroppingFormatCroppedCircle,
  kQ3FisheyeCroppingFormatFullFrame,
  kQ3FisheyeCroppingFormatSize32 = 0x7FFFFFFF
} TQ3FisheyeCroppingFormat;

Constants

kQ3FisheyeCroppingFormatCircular
The fisheye rendering fills a circular area that fits tightly in the smaller view dimension and is centered in the larger view dimension.
kQ3FisheyeCroppingFormatCroppedCircle
The circular area of fisheye rendering is tight against the left and right edges of the view, and is cropped on the top and bottom.
kQ3FisheyeCroppingFormatFullFrame
The fisheye rendering is cropped so that the view rectangle is totally filled.

TQ3FisheyeMappingFunction

Type of fisheye mapping function. See Wikipedia on Fisheye Lenses. 
typedef enum TQ3FisheyeMappingFunction
{
  kQ3FisheyeMappingFunctionOrthographic = 0,
  kQ3FisheyeMappingFunctionStereographic,
  kQ3FisheyeMappingFunctionEquidistant,
  kQ3FisheyeMappingFunctionEquisolidAngle,
  kQ3FisheyeMappingFunctionSize32 = 0x7FFFFFFF
} TQ3FisheyeMappingFunction;
Discussion

The mapping function relates the angle of an incoming light ray to the distance r from the center of the sensor to the image of the ray, taking into account the focal length f.

Constants

kQ3FisheyeMappingFunctionOrthographic
r = f sin( angle )
Since this function is increasing only up to an angle of pi/2, this kind of fisheye lens cannot have an angle of view greater than pi (180 degrees).
kQ3FisheyeMappingFunctionStereographic
r = 2 * f * tan( angle/2 )
kQ3FisheyeMappingFunctionEquidistant
r = f * angle
kQ3FisheyeMappingFunctionEquisolidAngle
r = 2 * f * sin( angle/2 )
