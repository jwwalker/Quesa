TQ3ErrorMethod

typedef Q3_CALLBACK_API_C(void, TQ3ErrorMethod)( TQ3Error firstError, TQ3Error lastError, TQ3Int32 userData);

Error Manager error callback.

Parameters

NameDescription
firstErrorThe first error which has occurred since the last time the handler was invoked.
lastErrorThe most recent error which has occurred.
userDataThe reference parameter passed to Q3Error_Register.
