Q3GroupIterator


Abstract

Class to simplify iterating through a Quesa group.

Discussion

Example: 


Q3GroupIterator		iter( aGroup, kQ3ShapeTypeGeometry );
CQ3ObjectRef	theItem;
while ( (theItem = iter.NextObject()).isvalid() )
{
// do something with theItem...
}
