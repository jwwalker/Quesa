TQ3FileModeMasks

typedef enum TQ3FileModeMasks {
    kQ3FileModeNormal                           = 0,
    kQ3FileModeStream                           = (1 << 0),
    kQ3FileModeDatabase                         = (1 << 1),
    kQ3FileModeText                             = (1 << 2),
    kQ3FileModeSwap                             = (1 << 3),
#endif
    kQ3FileModeSize32                           = 0xFFFFFFFF
} TQ3FileModeMasks;
Discussion

Mode masks for file operations.

Constants

kQ3FileModeNormal
Normal file mode.
kQ3FileModeStream
Stream mode.
kQ3FileModeDatabase
Database mode.
kQ3FileModeText
Text mode.
kQ3FileModeSwap
Binary mode with byte order swapped (not present in QD3D).
