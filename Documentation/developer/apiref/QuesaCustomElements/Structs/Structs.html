TCEUrlData

typedef struct TCEUrlData {
    char                                        *url;
    char                                        *description;
    TCEUrlOptions                               options;
} TCEUrlData;
Discussion

Describes the data for a URL.

Field Descriptions
url
C string containing the full (scheme + path) URL.
description
Human readable description of the URL.
options
Options for the URL.
