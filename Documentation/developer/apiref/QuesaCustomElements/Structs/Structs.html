TCEHalfspaceFogData

Data specifying half-space fog. 
typedef struct TCEHalfspaceFogData {
    float                   rate;
    TQ3RationalPoint4D      plane;
} TCEHalfspaceFogData;
Discussion

A point P is in the foggy halfspace if the dot product of P and the plane is negative. For example, if you want fog to exist where y < 5, you could use a plane (0, 1, 0, -5), and to have fog where y > 5, you could use (0, -1, 0, 5).

Field Descriptions
rate
A measure of the rate of increase in fog as you move farther from the plane.
plane
A plane defining a halfspace in world coordinates, see discussion.
