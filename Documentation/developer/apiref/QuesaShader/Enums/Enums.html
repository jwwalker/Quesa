Constants

kQ3ElementTypeTextureShaderAlphaTest
Normally, geometry using a texture with an alpha channel must be broken into triangles to be sorted by depth and rendered back to front. With this element, you can instead treat each texel as either completely opaque or completely transparent, avoiding depth sorting. The element value is a depth threshold, i.e., alpha less than the threshold is considered completely transparent. Data type: TQ3Float32.
