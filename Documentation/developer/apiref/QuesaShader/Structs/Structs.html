TQ3TextureUploadPropertyData

Data of a property of type kQ3TexturePropertyCallbackAfterUpload. 
typedef struct TQ3TextureUploadPropertyData {
    TQ3TextureUploadedCallback _Nonnull  callback;
    void*  _Nullable          context;
} TQ3TextureUploadPropertyData;
Discussion

Note that both fields of the structure are pointers, whose sizes depend on whether you are compiling a 64-bit binary.

Field Descriptions
callback
A callback function in your code.
context
A pointer that the engine will pass back to your callback function.
