TQ3ViewStatus

typedef enum TQ3ViewStatus {
    kQ3ViewStatusDone                           = 0,
    kQ3ViewStatusRetraverse                     = 1,
    kQ3ViewStatusError                          = 2,
    kQ3ViewStatusCancelled                      = 3,
    kQ3ViewStatusSize32                         = 0xFFFFFFFF
} TQ3ViewStatus;
Discussion

View submit loop status.

Constants

kQ3ViewStatusDone

kQ3ViewStatusRetraverse
Submit loop requires another pass.
kQ3ViewStatusError
Submit loop encountered an error.
kQ3ViewStatusCancelled
