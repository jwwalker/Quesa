TQ3ComputeBounds

typedef enum {
    kQ3ComputeBoundsExact                       = 0,
    kQ3ComputeBoundsApproximate                 = 1
} TQ3ComputeBounds;

Bounds accuracy types.

Constants

NameDescription
kQ3ComputeBoundsExactCompute exact bounds by transforming vertices to world coordinates then calculating bounds. Slower, but more accurate.
kQ3ComputeBoundsApproximateCompute approximate bounds by transforming local bounding box to world coordinates. Faster, but less accurate.
