CQ3ObjectRef_Gets.h



This header provides C++ wrappers for Quesa functions that return an object reference in an output parameter, as a convenience. In failure conditions, the CQ3ObjectRef may contain NULL. The name of the wrapper is the name of the original function with a C prepended.
