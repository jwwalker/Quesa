TQ3RendererPassInfo

Pass data in the kQ3RendererPropertyPassType property that the renderer may set on the renderer object. 

struct TQ3RendererPassInfo {
    TQ3RendererPassType        passType;
    TQ3LightObject _Nullable    light;
} ;
Discussion

When you obtain this property, you do not get a new reference to the light, so do not dispose it.

Field Descriptions
passType
Type of the current pass.
light
Current shadowing light, or nullptr if this is a pass of type kQ3RendererPassNonShadow.
