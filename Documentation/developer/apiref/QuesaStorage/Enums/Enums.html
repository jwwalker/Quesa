TQ3StorageOpenness

typedef enum TQ3StorageOpenness
{
  kQ3StorageOpenness_Ignored = 0,
  kQ3StorageOpenness_Closed,
  kQ3StorageOpenness_Open
} TQ3StorageOpenness;
Discussion

Open or closed state of a storage object. Be aware that some types of storage object do not have separate open and closed states, so this enumeration has 3 values.

Constants

kQ3StorageOpenness_Ignored
The storage ignores open and close commands, as in for example memory storage.
kQ3StorageOpenness_Closed
The storage is currently closed.
kQ3StorageOpenness_Open
The storage is currently opened.
