This page contains several images rendered using Quesa.

Geom Test

The Geom Test application is included within the SDK/Examples folder of the Quesa distribution, and is portable to Mac OS (9 and X), Linux, and Windows.

These images show Quesa using Apple's RAVE-based renderer on Mac OS 9, and Quesa's OpenGL-based renderer on Mac OS X, Linux, and Windows.
