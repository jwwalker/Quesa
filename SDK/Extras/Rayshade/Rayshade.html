What is Rayshade for Quesa (QuickDraw3D)?

It is a raytracer plugin renderer for Quesa (QuickDraw3D). One of the best features of Quesa (QuickDraw3D) is plugin renderers, Rayshade is the proof of this concept.

This renderer supports the following features in any Quesa program that supports Quesa plugin renderers.

  • Real shadows
  • Transparency
  • Per-pixel phong shading
  • Adaptive antialiasing

Rayshade is an extensible system for creating ray-traced images. Rayshade for Quesa is a port of the Rayshade renderer, it works as a Quesa PlugIn renderer. The source code for this project is also available from Quesa CVS.

Related links

Installation:

  • Drop the file Rayshade to your System Folder:Extensions.

Release notes

Notes on Version 0.7 Released 7/8/2001

This is the last release do by me(boga@mac.com). From now on i hand over the development to the Quesa community, as i have really no time to continue this project.

Changes/fixes:

  • Ported to Windows, Carbonized, and integrated into the quesa SDK.

Known bugs/Limitations:

  • Still doesn't work with Quesa.
  • Has some problems with Design Works.
  • Background color is always black.

Notes on Version 0.6 Released 3/4/2000

Changes/fixes:

  • Works with Microspot 3D World, Form-Z.
  • No more text warnings.
  • Transparency supported.
  • Memory allocated in Temporary area.

Known bugs/Limitations:

  • Still doesn't work with Quesa.
  • Has some problems with Design Works.
  • Background color is always black.

Notes on Version 0.5 Released 2/27/2000

This is the initial release. The major problem with this release that it doesn't work with Quesa0.1.0. I'm going to fix it in the next release.

Known bugs/Limitations:

  • Doesn't work with Quesa.
  • Renders everything as triangles. (Slow)
  • No trasparency, reflection, refraction.
  • Can't turn antialiashing off.

ToDo:

    • Soft shadows
    • Reflections
